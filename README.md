# 剑桥少儿英语拼写练习

为剑桥 Pre A1 Starters、A1 Movers 和 A2 Flyers 级别学生设计的单词拼写练习应用。

## 功能特性

- **中文提示拼写** - 显示中文意思，输入英文单词
- **虚拟键盘** - 点击字母输入，适合触屏设备
- **物理键盘支持** - 直接用键盘 a-z 输入，Enter 提交，Backspace 删除
- **错词本** - 自动记录答错的单词，支持错词复习模式
- **学习进度** - 统计练习次数、正确率，本地存储不丢失
- **清除记录** - 一键清除所有学习数据

## 快速开始

```bash
cd frontend/public
python3 -m http.server 8001
```

访问 http://localhost:8001 开始练习。

## 项目结构

```
frontend/public/
├── index.html        # 入口页面
├── practice.html     # 主练习页面
├── vocabulary.js     # 词汇库
├── css/
│   └── practice.css  # 样式
└── js/
    └── practice.js   # 交互逻辑
```

## 技术栈

- 前端：HTML + CSS + JavaScript（无框架）
- 存储：浏览器 localStorage
- 服务：Python http.server（或其他静态服务器）

## 快捷键

| 按键 | 功能 |
|------|------|
| A-Z | 输入字母 |
| Enter | 提交答案 / 下一个单词 |
| Backspace | 删除字母 / 重新输入 |
| Space | 下一个单词（提交后） |
| ESC | 关闭弹窗 |
